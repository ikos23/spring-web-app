Available Services

List of all employees

This service returns list of employees in JSON format. If XML format is required - just replace the last part of the URI json with xml.

List of mentorship programs

This service returns list of mentorship programs in JSON format. If XML format is required - just replace the last part of the URI json with xml.

List of all phases

This service returns list of all phases in JSON format.

Reference data service

This service can be used to get different reference data. Supported formats are json, xml and html. To switch format just change URI (e.g. ..../all/json/... to ..../all/xml/...). Available ref data : statuses and roles (URI end)

List of logs

Returns some logs in XML format.

Get assignments

Assignment is an Employee-Role relationship for particular phase.

Get mentees with overall duration in programs (in days)

This service returns some information about mentees.
